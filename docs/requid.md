
# 开源数字标牌（Digital Signage）/多屏内容分发系统方案

本项目旨在基于 [xplay.io](https://github.com/wendal/xplay.io) 开源代码，实现一套 Android + 服务端 的数字标牌系统，适用于跨门店、写字楼、校园等多种场景，具备以下能力：

---

## 1️⃣ 终端侧 (Android / APK) 能力

**用途:** 安装在显示器、电视、Android 盒子或平板上的客户端应用。

### 基础功能
- 原生 Android APK（兼容 Android TV / 盒子 / 平板）
- **全屏内容播放**，高性能渲染
- 支持内容类型：
  - 图片（JPG/PNG）
  - 视频（MP4/MKV 等）
- **自动轮播**（可配置播放顺序与各媒体播放时长）
- **开机自启**（保证通电即播，适合无人值守环境）
- **断网可正常播放**（本地缓存内容，临时离线不影响播放）

### 管理与控制
- 唯一设备标识（Device ID/绑定码/二维码）
- 启动后自动与服务端连接
- 支持服务端下发播放内容和计划，自动接收更新
- 远程内容刷新与更新
- 可选能力：远程截屏、心跳上报、在线/离线状态反馈

> **关键词:**  
> Android Signage Player、数字标牌 Android 客户端、离线轮播、设备绑定

---

## 2️⃣ 服务端（统一管理后台）能力

**用途:** 集中式管理所有设备和内容，统一下发播放计划。

### 内容管理
- 图片/视频上传（支持批量）
- 内容分组、标签管理
- 播放列表（Playlist）自定义配置

### 设备管理
- 设备注册绑定（扫码/ID）
- 设备分组（如门店/区域/楼层归类）
- 查看设备在线/离线状态 & 信息

### 策略调度
- 面向单设备/组，分发内容和播放计划
- 定时播放，支持时间段切换内容（如白天A，晚上B）
- 内容更新自动推送

### 技术形态与推荐技术栈
- Web 管理后台（React/Vue/Ant Design Admin）
- REST API / WebSocket / MQTT（终端与服务端通讯）
- 典型后端：Java/Spring Boot、Node.js、Python/Django
- 数据库推荐：MySQL / PostgreSQL

> **关键词:**  
> Signage Server、内容管理、数字标牌 CMS、设备监控、统一调度
